# Regionernas läkemedelskostnader

Det här är beräkningarna och materialet bakom Dagens Samhälles artikel om regionernas stegrande läkemdelskostnader. Källor är SKL för siffror på läkemdelskostnader och SCB för korrekt konvertering till fasta priser under perioden efter relevant prisindexering. Övrig info och relevanta länkar finns i notebooken.
